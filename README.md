To run the application simply run the Dungeon_of_Solitude.exe file.
A pop up will appear. Select 1280x720 for the screen resolution and fantastic for the graphics quality then click play.
Use Alt + F4 to exit the game if using windows or Command + Q if using mac.
To open the project in Unity simply select open at the unity start screen and locate the 2DGame_FYP folder and select it.
